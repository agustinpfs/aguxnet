Comandos Tutorial de Ubuntu Server

Visitar video tutorial

Conexión al servidor de manera remota

ssh nombreDeUsuarioDelServidor@ip(cable o wifi)

Ejemplo:

ssh agux@192.168.0.228

Configuración ip fija

Ver nombre de interfaces:

ip a

Ver nombre de archivo "Netplan":

ls /etc/netplan/

Editar archivo "Netplan":

sudo nano /etc/netplan/nombreDelArchivo

Ejemplo:

sudo nano /etc/netplan/50-cloud-init.yaml

*Eliminar configuración por defecto (ctrl k)

Pegar la sig config:

network:
  version: 2

  ethernets:
    enp1s0:
      addresses:
        - 192.168.0.228/24
      routes:
        - to: default
          via: 192.168.0.1
      nameservers:
        addresses: [8.8.8.8, 1.1.1.1]

  wifis:
    wlp2s0:
      addresses:
        - 192.168.0.203/24
      access-points:
        "Fibertel WiFi077 2.4GHz":
          password: "84752097502"


⚠️Cambiar las ip y el nombre y contraseña de tu wifi
Para salir: ctrl x - y - enter

Actualizar archivo "Netplan":

sudo netplan apply

Apagar el servidor remotamente:

sudo poweroff

Salir de la conexión remota:

exit
