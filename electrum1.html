Comandos Linux en instalación de Electrum

Chek version gpg (verificación firmas)

gpg --version

Guía para la verificación de firmas

Instalación

Comando si tenés el error de LIBFUSE:

sudo apt-get -y install libfuse2

Para agregar acceso directo e ícono

Crear archivo electrum.desktop:

En Home/.local/share/applications/ :
nano electrum.desktop

Copiar dentro del archivo electrum.desktop:

[Desktop Entry]
Name=Electrum
Exec=/home/agustin/Electrum/electrum-4.5.5-x86_64.AppImage
Icon=/opt/Electrum/electrum-icon.png
Type=Application

ATENCIÓN: En "Exec" cambiar el nombre de home (agustin) y corroborar el nombre del archivo (electrum-4.5.5-x86_64.AppImage) por si cambió la versión.

Agregar imagen ícono:

sudo wget http://icons.iconarchive.com/icons/alecive/flatwoken/256/Apps-Electrum-icon.png

Cambiar nombre archivo icono (por el mismo que tengas en tu archivo electrum.desktop):

sudo mv Apps-Electrum-icon.png electrum-icon.png
